Overview

This talk was originally developed for the Master of Environmental Data Science (MEDS) program, administered by the Bren School of Environmental Science & Management in partnership with the National Center for Ecological Analysis and Synthesis.

Abstract

Our MEDS students started off the inaugural year by learning how to create their own personal websites using the {distill} R package. This workshop offers an intro to CSS and how to use it to further customize distill sites. We cover how to change the default element styling (fonts, colors, sizes), how to create custom CSS rules, and how to find inspiration from other distill websites (and their source code) on the distillery.
